# 📊 Ecommerce Sales Analysis (Excel Dashboard Project)

## 🔹 Project Overview

This project presents a comprehensive analysis of ecommerce sales data using Microsoft Excel. The objective is to extract actionable insights from raw transactional data and visualize them through interactive dashboards and charts.

The project demonstrates strong skills in data cleaning, transformation, and visualization using Excel.

---

## 🔹 Dataset Summary

* 📦 Total Orders: **5,009**
* 💰 Total Sales: **$2,297,200+**
* 📈 Total Profit: **$286,397+**
* 🔢 Total Quantity Sold: **37,873 units**
* 📅 Time Period: **2011 – 2014**

---

## 🔹 Tools & Techniques Used

* Microsoft Excel
* Pivot Tables & Pivot Charts
* Data Cleaning & Transformation
* Dashboard Design
* KPI Metrics & YoY Analysis
* Slicers for Interactivity

---

## 🔹 Dashboard Features

* 📊 Sales & Profit Overview
* 📈 Year-wise Sales Trend Analysis
* 🥇 Top 5 Products/Segments Analysis
* 🌍 Region-wise Performance (Map Chart)
* 🥧 Category Distribution (Pie Chart)
* 📉 Profit Analysis using Waterfall Chart
* 🎯 KPI Tracking (Sales, Profit, Growth)

---

## 🔹 Key Insights

* Identified high-performing product categories and regions
* Observed sales growth trends across multiple years
* Detected areas with high discounts impacting profitability
* Analyzed customer purchasing behavior and order patterns

## 🔹 How to Use

1. Download the Excel file
2. Open in Microsoft Excel (2016 or later recommended)
3. Navigate to the **Dashboard** sheet
4. Use slicers to interact with data
